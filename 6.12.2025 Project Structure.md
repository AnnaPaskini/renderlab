## File Organization Principles

1. **App Router**: All routes in `app/` directory
2. **Component Colocation**: Components near their usage
3. **Utility Separation**: Core utilities in `lib/` and `core/`
4. **Type Safety**: Strict TypeScript configuration
5. **Modular Architecture**: Feature-based organization
