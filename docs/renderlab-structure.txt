Project Overview:
================
RenderLab is a Next.js application built with TypeScript and Tailwind CSS.
It appears to be a design/rendering platform with authentication, workspace
functionality, and marketing pages.

Key Features:
- Authentication system (login/signup)
- Marketing website with pricing and blog
- Workspace interface for image/design work
- Component library with UI elements
- API endpoints for generation and history
- Responsive design with mobile/desktop navigation

Tech Stack:
- Next.js (React framework)
- TypeScript
- Tailwind CSS
- Custom component library
- File-based routing
- API routes
